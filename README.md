# my-blog

基于`Nuxt`实现的服务端渲染，博客系统。
# 预览地址

[我的博客](estding.com)

> Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install*[see note below]

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```
